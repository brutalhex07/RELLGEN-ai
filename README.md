# AI1 Studio - Firebase Studio Prototype

This is a premium AI creativity suite prototype built with Next.js, Genkit, and Shadcn UI.

## 🚀 Quick Access URLs

### User Application
- **Homepage:** `/`
- **Login:** `/auth/login`
- **Sign Up:** `/auth/signup`
- **Dashboard:** `/dashboard`
- **Image Studio:** `/dashboard/images`
- **AI Chat:** `/dashboard/chat`

### Admin Console
- **Admin Dashboard:** `/admin/dashboard`
- **User Management:** `/admin/users`
- **Revenue Tracking:** `/admin/payments`

## 🛠️ Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS + Shadcn UI
- **AI Engine:** Google Genkit + Gemini 2.0 Flash / Imagen 3
- **Icons:** Lucide React
- **Charts:** Recharts

## 💻 Development
To start the development server:
```bash
npm run dev
```
The app will be available at `http://localhost:9002`.
