# **App Name**: AI1 Studio

## Core Features:

- AI Conversational Agent: Engage with advanced AI models (like Gemini) through a chat interface, generating textual responses to user prompts.
- Generative Image Tool: Create unique images from textual descriptions using AI, with selectable artistic styles (e.g., realistic, anime).
- Secure User Authentication: Allows users to sign up and log in securely using email/password or Google accounts.
- Interaction History: Stores and displays past AI chat conversations and generated images for easy access and review.
- Credit & Usage Management: A basic system to track user credits and consumption for AI interactions, integrating with a simple purchasing flow.
- Intuitive User Dashboard: A centralized dashboard with sidebar navigation for easy access to all AI tools, history, and account settings.
- Content Download Options: Provides a clear option to download generated images directly from the application.

## Style Guidelines:

- Color scheme: Dark theme, using a subtle gradient of deep blues and purples. The general feel is modern, premium, and sophisticated.
- Primary foreground elements: A vibrant, clear blue-purple, hex #844DF0 (HSL 260, 70%, 60%), providing contrast and energy.
- Background: A very dark, slightly purple-tinted grey, hex #1C1921 (HSL 260, 15%, 12%), serving as a sophisticated canvas.
- Accent: A bright, electric blue, hex #1464FF (HSL 230, 80%, 50%), used for interactive elements and highlights to create dynamic visual interest.
- Headline and body font: 'Inter' (sans-serif) for its modern, clean, and highly readable appearance, suitable for a professional SaaS application.
- Use minimal, crisp, outlined icons with a subtle glassy or gradient effect to align with the glassmorphism UI concept.
- Implement a glassmorphism-inspired UI with semi-transparent, layered elements that create depth and a modern aesthetic, with a persistent left sidebar and dynamic main content area. Ensure full mobile responsiveness.
- Incorporate smooth, subtle animations for UI transitions, element hovers, and specific actions like AI typing responses or content loading, enhancing the premium feel without being distracting.
